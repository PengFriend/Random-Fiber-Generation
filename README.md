# Random Fiber Generation Project Description

## Runtime Environment
- **Operating System**: Windows 11
- **Execution Method**: Direct execution of the executable file

## Quick Start
1. Download the project files
2. Double-click to run `Random Fiber Generation.exe`
3. Enter the parameters according to the prompts
